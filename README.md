# 500일 꿈속 호텔

모바일에서 플레이하는 픽셀아트 스타일의 500일 기념 웹게임 초안입니다.

## 현재 구현

- 호텔 타이틀 화면과 로비
- 308호 플레이 가능한 초안
  - 잠실 헬스장에서 율리 찾기
  - 첫 메시지 연출
  - 강남교보타워점 수저·나이프 봉투 포장
  - 전주 PNB 선반에서 말차 초코파이 고르기
  - 약국 선반에서 수상한 물약 고르기
  - 말차 딸기빙수에 마지막 딸기 올리기
  - `2025.03.08` 클리어 화면

## 실행

별도 설치 없이 `index.html`을 열면 실행됩니다. GitHub Pages를 활성화하면 휴대폰에서 링크로 플레이할 수 있습니다.

GitHub 저장소의 `Settings → Pages → Build and deployment → Deploy from a branch`에서 `main / (root)`를 선택하면 됩니다.

## 프로젝트 구분

이 저장소는 개인 500일 기념 웹게임 프로젝트이며, `pumpkin` 한이음 프로젝트와 별개입니다.
